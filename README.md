# Hash hush

## サイト概要

各々が好きな物事を共有する SNS

### サイトテーマ

嫌な情報の入ってこない自分の好きなことで溢れた SNS

### テーマを選んだ理由

私は SNS で様々な投稿や情報を見るのが好きで、暇な時にはよく SNS をチェックしています。
しかし中には、うんざりするような嫌な話題や悲しい話、炎上なども散見され、楽しいばかりではありません。
そこで、見たくない話題の入ってにくい、炎上しにくい SNS があったらいいなと思い、このテーマにしました。

### ターゲットユーザ

罵詈雑言の飛び交う SNS に嫌気のさした若い層

### 主な利用シーン

好きなことを共有したいと思ったその時
